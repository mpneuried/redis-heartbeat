¶ RedisConnector

¶ extends NPM:MPBasic

¶ Exports: Class

Basic module to handle a redis connection. Just call @connect() within the constructor to connect savely to redis

¶ Class-Vars

  • redis RedisClient the generated redis client instance
  • connected Boolean Flag to mark if the module is currently connected to redis

  • redis:error: emitted on redis error.
